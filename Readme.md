# Data Science Assessment - Raghav Gaur

## Overview
This repository contains the analysis of trader behavior relative to the Fear & Greed Index.

## Structure
- `notebook_1.ipynb`: The Google Colab notebook containing all data cleaning and analysis code.
- `csv_files/`: Contains raw and processed datasets.
- `outputs/`: Contains the generated visualization charts.
- `ds_report.pdf`: The final summary report of findings.

## Key Insights
- Traders are most profitable during "Extreme Greed" market conditions.
- There is a strong correlation between "Fear" sentiment and increased buying activity.
